Personal Portfolio

Welcome to my personal portfolio website! This project showcases my skills, experience, and selected works in a clean, modern, and responsive design.

Features

- Personal Introduction: A brief overview about me, including my photo and CV.
- Project Gallery: Interactive cards highlighting my key projects, each with a title, description, link, and category.
- Responsive Design: Optimized for viewing on desktops, tablets, and mobile devices.
- Easy Deployment: Ready to be deployed on platforms like Netlify or Vercel.

Getting Started

To personalize and deploy this portfolio:

1. Update Your Photo:  
   Replace avatar.jpg with your own professional photo.

2. Add Your CV:  
   Place your updated CV as cv.pdf in the project root.

3. Customize Projects:  
   Edit index.html to update the project cards with your real project titles, descriptions, links, and categories.

4. Style Adjustments (Optional):  
   Modify styles.css to further personalize the look and feel.

5. Deploy:  
   Upload the project folder to a static hosting service such as Netlify or Vercel.

File Structure

- index.html – Main webpage structure
- styles.css – Custom styles for the site
- script.js – JavaScript for interactivity
- avatar.jpg – Your profile photo
- cv.pdf – Your CV/Resume
- README.txt – Project instructions
